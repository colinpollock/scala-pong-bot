README
======

Keeps track of the results of challenge games and updates each player's rank
after a game.


Functionality
-------------


!pong leaders
!pong robovoyo defeated rpidikiti 21-18


Possibly
~~~~~~~~
!challenge robovoyo
!accept challenge from xlorm



PARAMS
------
* Number of provisional games.
* Default starting rank.
* ELO params.
